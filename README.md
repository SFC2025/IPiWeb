# Nacho Pirola Web

Demo web de catálogo mayorista para la marca Nacho Pirola.

El proyecto presenta una experiencia moderna para visualizar productos, explorar categorías y generar pedidos de manera rápida mediante WhatsApp.

## Tecnologías usadas

* HTML5
* CSS3
* JavaScript
* GitHub Pages

## Funcionalidades

* Catálogo visual de productos
* Filtros por categoría
* Buscador integrado
* Carrito lateral interactivo
* Cálculo automático de subtotal
* Aplicación de descuentos
* Generación de pedido por WhatsApp
* Diseño responsive para desktop y dispositivos móviles

## Estructura del proyecto

* `index.html`
* `style.css`
* `script.js`
* `/img`

## Estado del proyecto

Versión demo en desarrollo.

Las imágenes, productos, precios, categorías y contenido pueden modificarse según las necesidades de la marca.
